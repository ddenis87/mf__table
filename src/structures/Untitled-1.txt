

serialization()
  Цикл по секциям settings (пропускаем те что вложенны)
    Записать в ObjectJSON( serializationArea(имяСекции) )
  Сохранить ObjectJSON

serializationArea(имяСекции)
  Получить параметры секции из settings по имени
  Получить диапазон секции
  Получить данные по диапазону

  Если секция массив
    Цикл по данным
      Добавить секцию с данными во Временное размещение
      Если есть вложенная секция
        Добавить в секцию во Временное размещение SerializationArea(имяВложеннойСекции)

  Если секция объект
    Записать данные во Временное размещение по имени секции
    Если есть вложенная секция
        Добавить в секцию во Временное размещение SerializationArea(имяВложеннойСекции)

  Вернуть Временное размещение


serializationArea(имяСекции)
  Получить параметры секции
  Временное размещение [] или {}
  Получить данные секции
  Записать во Временное размещение данные
  Если есть вложенная секция
    Получить вложенные секции
    Цикл по вложенным секциям
      Записать во Временное размещение serializationArea(имяВложеннойСекции)
  Вернуть Временное размещение

serializationNestedArea(имяСекции, имяСекцииРодителя = (по умолчанию весьДокумент))
  Получить параметры имяСекции
  Временное размещение [] или {}
  Получить все секции с имяСекции в диапазоне имяСекцииРодителя
  Цикл по секция
    Получить данные секции
    Записать во Временное размещение
    Если есть вложеннаяCекция
      Записать во Временное размещение serializationNestedArea(вложеннаяCекция, секция)
  Вернуть Временное размещение
